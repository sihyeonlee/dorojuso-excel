# dorojuso-excel

## About
dorojuso-excel은 정해진 엑셀 서식에 학교 이름을 입력하면, 서울시 지역 내에 도로명 주소를 검색하여 반환합니다.  
사회복무요원으로 복무 중, 상장을 송부하는 과정에서 매번 학교 주소를 검색하여 입력하기가 번거로워 작업하였습니다.  
도로명 주소 open api를 사용하여, 입력한 학교 이름을 검색하여 리턴된 결과 값을 엑셀에 저장합니다.
